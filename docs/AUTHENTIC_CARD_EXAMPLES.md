# Authentic Vietnamese Lô Tô Card Examples

This document shows visual examples of authentic Vietnamese Lô Tô cards generated by the game engine.

## Card Format

- **3 rows × 9 columns** = 27 cells total
- **15 numbers** + **12 blanks** per card
- **Each row: exactly 5 numbers + 4 blanks**
- **Numbers range: 1-90** (NOT 0-99)
- **Column constraints:**
  - Column 0: 1-9
  - Column 1: 10-19
  - Column 2: 20-29
  - Column 3: 30-39
  - Column 4: 40-49
  - Column 5: 50-59
  - Column 6: 60-69
  - Column 7: 70-79
  - Column 8: 80-90

## Win Condition

In authentic Vietnamese Lô Tô, **only ONE win condition exists:**

**Complete any horizontal row (all 5 numbers marked) = WIN**

Example winning row:
```
┌────┬────┬────┬────┬────┬────┬────┬────┬────┐
│ [3]│    │[25]│    │[47]│    │    │[72]│    │  ← ROW WIN! All 5 numbers called
├────┼────┼────┼────┼────┼────┼────┼────┼────┤
│    │ 12 │    │ 33 │    │ 56 │ 68 │    │ 89 │
├────┼────┼────┼────┼────┼────┼────┼────┼────┤
│  7 │    │ 21 │    │ 44 │    │ 61 │    │ 90 │
└────┴────┴────┴────┴────┴────┴────┴────┴────┘

Called numbers: 3, 25, 47, 72, ... (needs one more)
When the 5th number in row 0 is called → Player shouts "Kinh!" or "Lô tô!"
```

- More cards = higher chance of winning
- Numbers CAN repeat across different cards (they're independent)
- Player must watch ALL their cards simultaneously

## Key Differences from American Bingo

| Feature | American Bingo | Vietnamese Lô Tô |
|---------|---------------|------------------|
| **Grid Size** | 5×5 = 25 cells | 3×9 = 27 cells |
| **Numbers** | 75 (1-75) | 90 (1-90) |
| **Numbers per Card** | 24 + 1 FREE | 15 |
| **Blanks** | 1 FREE space | 12 blanks |
| **Column Rules** | B=1-15, I=16-30, etc. | Col 0=1-9, Col 1=10-19, etc. |
| **Row Numbers** | 5 per row | 5 per row |
| **Win Conditions** | Line, 4 corners, X, full card | **ONE ROW ONLY** |
| **Cultural Context** | Church fundraisers | Tết (Lunar New Year) celebrations |

## Traditional Calling Phrases

In Vietnamese culture, callers often use rhyming phrases:

- **11** - "Hai cây giậu" (Two sticks)
- **22** - "Hai con ngỗng" (Two geese)
- **69** - "Mười chín nụ hôn" (Nineteen kisses)
- **88** - "Hai bà già" (Two old ladies)

These phrases add entertainment and help players verify numbers were heard correctly.
